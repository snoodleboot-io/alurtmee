Grab Alurtmee.

Runs on Linux today — macOS and Windows are planned. The list below is pulled live from GitHub, so it always shows the current release and the last four.

Before you run it: Alurtmee renders with your GPU and stores your token in the system keychain, so you'll need GPU drivers (mesa-vulkan-drivers) and a Secret Service daemon (gnome-keyring) present. The AppImage needs no install — chmod +x and run. The .deb pulls in its runtime dependencies via sudo apt install ./alurtmee_*_amd64.deb. First look? Launch with ALURTMEE_DEMO=1 for sample data and no network.
Verify any download. Every release ships a SHA256SUMS file and a signed build-provenance attestation: sha256sum -c SHA256SUMS checks integrity, and gh attestation verify Alurtmee.AppImage --repo snoodleboot-io/alurtmee confirms this repo's CI built it.
